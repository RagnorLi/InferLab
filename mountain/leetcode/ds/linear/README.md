# 线性结构 (Linear Structures)

**特征**：元素之间是一对一关系，每个元素最多有一个前驱和一个后继。

## 数据结构清单

```
线性结构
│
├── 数组类 (Array-based)
│   ├── 数组 (Array)
│   │   └── 连续内存，随机访问 O(1)
│   ├── 动态数组 (Vector / Dynamic Array)
│   │   └── 可自动扩容的数组
│   └── 字符串 (String)
│       └── 字符数组
│
├── 链表类 (Linked List)
│   ├── 单链表 (Singly Linked List)
│   │   └── 单向链接的节点序列
│   ├── 双链表 (Doubly Linked List)
│   │   └── 双向链接的节点序列
│   └── 循环链表 (Circular Linked List)
│       └── 首尾相连的链表
│
└── 受限线性表 (Restricted Linear List)
    ├── 栈 (Stack)
    │   └── LIFO 后进先出
    ├── 队列 (Queue)
    │   └── FIFO 先进先出
    └── 双端队列 (Deque)
        └── 两端都可以进出
```

**总计：9 种数据结构**
