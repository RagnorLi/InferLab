# 图结构 (Graph Structures)

**特征**：元素之间是多对多关系，形成网络结构。

## 数据结构清单

```
图结构
│
├── 存储表示 (Representation)
│   ├── 邻接表 (Adjacency List)
│   │   └── 用链表数组表示图
│   └── 邻接矩阵 (Adjacency Matrix)
│       └── 用二维数组表示图
│
└── 图类型 (Graph Type)
    ├── 无向图 (Undirected Graph)
    │   └── 边无方向
    ├── 有向图 (Directed Graph)
    │   └── 边有方向
    └── 加权图 (Weighted Graph)
        └── 边有权重
```

**总计：5 种数据结构**
