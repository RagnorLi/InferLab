# 图结构 (Graph Structures)

**特征**：元素之间是多对多关系，形成网络结构。

## 数据结构清单

| 序号 | 数据结构 | 英文名 | 说明 |
|------|---------|--------|------|
| 1 | 图（邻接表） | Graph (Adjacency List) | 用链表数组表示图 |
| 2 | 图（邻接矩阵） | Graph (Adjacency Matrix) | 用二维数组表示图 |
| 3 | 无向图 | Undirected Graph | 边无方向 |
| 4 | 有向图 | Directed Graph | 边有方向 |
| 5 | 加权图 | Weighted Graph | 边有权重 |

**总计：5 种数据结构**
