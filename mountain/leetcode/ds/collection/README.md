# 集合结构 (Collection Structures)

**特征**：元素之间无显式结构关系，通过映射函数组织数据。

## 数据结构清单

```
集合结构
│
├── 哈希类 (Hash-based)
│   ├── 哈希表 (Hash Table)
│   │   └── 基于哈希函数的键值映射
│   ├── 哈希映射 (HashMap)
│   │   └── 键值对集合
│   └── 哈希集合 (HashSet)
│       └── 无序元素集合
│
├── 树类 (Tree-based)
│   ├── 树映射 (TreeMap)
│   │   └── 基于树的键值映射（有序）
│   └── 树集合 (TreeSet)
│       └── 基于树的元素集合（有序）
│
└── 特殊集合 (Special Collections)
    ├── 并查集 (Union-Find / Disjoint Set)
    │   └── 集合的合并与查询
    └── 位集合 (BitSet)
        └── 用位向量表示集合
```

**总计：7 种数据结构**
