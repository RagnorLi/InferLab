# 树形结构 (Tree Structures)

**特征**：元素之间是一对多关系，形成层级结构，每个元素最多有一个父节点。

## 数据结构清单

```
树形结构
│
├── 二叉树类 (Binary Tree)
│   ├── 二叉树 (Binary Tree)
│   │   └── 每个节点最多两个子节点
│   ├── 二叉搜索树 (Binary Search Tree / BST)
│   │   └── 有序的二叉树
│   ├── AVL 树 (AVL Tree)
│   │   └── 自平衡二叉搜索树
│   ├── 红黑树 (Red-Black Tree)
│   │   └── 自平衡二叉搜索树
│   └── 堆 (Heap)
│       └── 完全二叉树，用于优先级队列
│
├── 多叉树类 (Multi-way Tree)
│   ├── N 叉树 (N-ary Tree)
│   │   └── 每个节点可以有 N 个子节点
│   └── B 树 (B-Tree)
│       └── 多路平衡搜索树
│
└── 特殊应用树 (Specialized Tree)
    ├── 前缀树 (Trie)
    │   └── 字符串检索树
    ├── 线段树 (Segment Tree)
    │   └── 区间查询与更新
    └── 树状数组 (Fenwick Tree / BIT)
        └── 前缀和查询
```

**总计：10 种数据结构**
